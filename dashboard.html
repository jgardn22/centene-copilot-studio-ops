The real story: zero purchased capacity, material workload

MSXI Licenses by Enrollment confirms Centene has no Copilot Credit P3, no Credit Pack, no Microsoft Agent P3. Yet they consume 29.5K Copilot Credits/mo + 1.2M Copilot Studio agent credits/mo. This re-frames the Dec 2025 peak from "oversized commitment" (the original hypothesis) to "demonstrated workload running against built-in pools" — which is a commercial conversation, not a usage investigation.

What's covering the consumption

  • 10× M365 Copilot Sub Add-on (Part 83I-00001). Includes built-in M365 Copilot agentic Copilot Credit allowance.
  • 2× Copilot Studio Legacy USL (Part SYS-00001). Authoring licenses for makers, NOT runtime capacity.
  • Tenant default credit pool. Covers low-volume Copilot Studio usage before metering kicks in.
  • 25× Microsoft Agent 365 Frontier Extension (free provisioning, AAS-73476). Not a billed entitlement.

Right-sizing options

  • Copilot Credit P3 Tier 1: 300,000 credits/yr, 5% discount, ~$28,500 net. Covers ~10 months of current burn (29.5K × 12 = 354K projected annual demand).
  • Copilot Credit P3 Tier 2: 1.5M credits/yr, 6% discount. Covers projected demand 4× over with headroom for the Dec-style peak (131K spike).
  • Copilot Studio License (Credit Pack): 25K credits/mo, $200/pack/mo (~$2,400/yr). Monthly subscription, no rollover. Closest fit to current 26K MTD.
  • Microsoft Agent P3 Tier 1: 20K ACU = 2M credits/yr, 5% discount. Foundry-inclusive; over-sizes Copilot Studio but adds room for broader agent platform usage.

Suggested next step: commercial conversation

Not "investigate the data" — talk to Centene about packaging

Centene is getting Copilot Studio value at the M365 Copilot built-in tier. The workload is steady enough (29K-35K credits/mo, peaks to 131K) that they would benefit from explicit capacity packaging, both for FinOps visibility and to avoid surprise PayGo charges if a flow suddenly scales. Best ask:

  • Talk to: Centene's IT FinOps / Copilot Studio admin (Dmytro Tluchak surfaced in recent licensing thread; brand-level: Brandon Callendar, Nick Barnhart)
  • Ask: "We're seeing ~30K Copilot Credits / month from your Power Automate workflows and Copilot Studio agents. As you scale, would Copilot Credit P3 (annual commit, discount) or Copilot Studio License (monthly pack) give you better FinOps control than the current M365 Copilot built-in pool?"
  • Why now: Centene already has 805 MAU on Copilot Studio C2 and 423 on Copilot Studio (full table on Operator tab). Adoption is locked in — only the commercial wrapper is missing.
